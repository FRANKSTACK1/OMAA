# OMAA Marketplace Landing Page

A clean, mobile-responsive landing page for the **OMAA** platform. Built for students, entrepreneurs, and businesses to showcase products and services.

## 🚀 Features
- **Responsive Design:** Optimized for mobile, tablet, and desktop screens.
- **Modern UI:** Built with Tailwind CSS for a sleek, orange-themed aesthetic.
- **User-Centric Layout:** Dedicated sections for Vendors and Buyers.
- **Call-to-Action:** Clear conversion paths for "Start Selling" and "Join Omaa".

## 🛠 Tech Stack
- **HTML5**
- **Tailwind CSS** (via CDN)
